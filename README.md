# Project_Basic_OpenCV
Minimum project to OpenCV C++ using cmake. 

## Need:
    Cmake installed.

## To compile:

1 - Go to folder build.
    cd build
2 - Run cmake, to build the flags to compiler.
    cmake ..
3 - Run the compiler.
    make
4 - Run the default application.
    ./example <your image>

    or

    ./example
    to open a default image.
